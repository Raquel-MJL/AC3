División principal

La etiqueta <div> viene de división, prácticamente funciona y sirve de contenedor de bloque, es decir puede contener uno ovarios elementos HTML dentro de ella, es bastante utilizada para agrupar vario elementos que luego serán apuntados con CSS con la finalidad de dar un estilo adecuado a dicho grupo, o también puede servir para organizar mejor nuestra página web

Atributos que se pueden usar con la etiqueta <div>

  • ID
  • CLASS
  • WIDTH
  • HEIGHT
  • STYLE
  • TITLE

¿Cuándo se usa la etiqueta div?

La aparición de HTML5 y sus nuevas etiquetas semánticas ha provocado que la etiqueta de división div se utilice cada vez menos en nuestras páginas webs.
